# log_session — Templates

| Template | Purpose |
|----------|---------|
| `report-template.md` | Structure for `reports/session-efficiency/efficiency-report-YYYY-MM-DD.md` |

## Output location

All generated files go to `reports/session-efficiency/`:
- `efficiency-report-YYYY-MM-DD.md` — one file per run, date in name
- `efficiency-benchmarks.md` — updated in place after every run
