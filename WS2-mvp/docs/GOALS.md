# Jio CX — Goals & Success Criteria

## 48-Hour Goal (Data Access + 2 Days)

**Deliverable:** One demo-ready finding from Jio's Databricks data.

**Success =** Mark Farbrace can put it on a slide and say to Venturini/Jio: "This is what we found in 2 days. Imagine what 90 days looks like."

**Anti-goals:**
- NOT a generic dashboard
- NOT a repeat of what Jio already monitors
- NOT a capability demo ("look what our tool can do")
- YES a specific insight they didn't know, tied to revenue/churn/NPS

## 90-Day Goals (June 30 Deadline)

### Track 1: Network Improvement via Agentic AI
- Identify the top 5 network event patterns that predict customer harm
- Build agentic detection → intervention pipeline (detect pattern → trigger proactive outreach)
- Measurable: reduction in complaint volume for pilot segment

### Track 2: Jio Buddy Prototype
- Working prototype: AI agent with memory + context, multi-channel (voice, chat, app)
- Core capabilities: network troubleshooting, plan optimization, proactive alerts
- Hindi + local language support
- Tested on small customer set across all major use cases
- Feeds into the 5 "Jio OS life modes" (career, city move, life event, parent, SME)

## Value Metrics (What Jio Measures)

| Metric | Current | Target Signal |
|--------|---------|---------------|
| MNP churn | 0.9% monthly | Identify predictive signals 14+ days before port request |
| Care contacts | 98% digital, 2% agent | Reduce the 2% further via proactive resolution |
| NPS | Tracked per journey | Identify which events drive NPS decline most |
| Complaint volume | Main leading indicator | Reduce via proactive intervention before complaint |
| Port-in vs port-out | Net positive | Maintain while growing ARPU |

## How Findings Feed Jio Buddy

Every analytical finding should map to a Jio Buddy capability:

| Finding | Jio Buddy Action |
|---------|-----------------|
| "X event pattern predicts churn" | Buddy detects pattern → proactive credit + "we noticed" message |
| "Silent sufferers have Y profile" | Buddy proactively reaches out to non-complainers with bad metrics |
| "Complaint fingerprint appears 48hrs before call" | Buddy intervenes before customer picks up phone |
| "Geography Z has worst experience" | Buddy adjusts expectations + offers alternatives in degraded areas |
| "Cross-sell readiness signal detected" | Buddy suggests ecosystem expansion at the right moment |
