# Basecamp Templates

## Available Templates

| Template | Purpose |
|----------|---------|
| `report-template.md` | Structure for basecamp orientation reports |
| `erd-template.mmd` | Placeholder Mermaid ERD (populated from live schema) |

## Report Naming

Reports are saved to `reports/basecamp/`:
- `basecamp-<project>-YYYY-MM-DD.md` — Full basecamp orientation report
